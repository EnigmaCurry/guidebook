# Changelog

## v0.1.0 — 2026-04-26

- Initial release of Guidebook, a personal webapp framework
- FastAPI backend with SQLAlchemy async + SQLite
- Svelte frontend with theme system
- Multi-database management with per-project settings
- Browser session authentication with self-signed TLS, JWT session tokens, configurable slots/TTL, and session transfer
- File attachments with upload, drag-drop, paste, image/video/audio preview, and lightbox
- Media gallery page with dual-pane view filtered by record search
- Ephemeral scratchpad with real-time sync across clients via SSE
- Keyboard navigation: arrow keys, Enter to edit, Escape to cancel, `/` to search
- Real-time cross-client updates via Server-Sent Events
- Auto-update from GitHub releases
- GitHub Actions CI/CD (Linux, macOS, Windows, Docker)
- SQL query interface
- Auto-backup system
- Notification system
